ترمكس : (بالإنجليزية: termux)‏ هو محاكي الطرفية، وهو تطبيق مفتوح المصدر يضيف بيئة اوامر لينكس على الهواتف دون الحاجة إلى ترويت الهاتف أو تثبيت نظام خارجي أو روم معدل، بل يكفي تثبيته مباشرة عبر جوجل بلاي حيث يوفر التطبيق العديد من الحزم الإضافية باستخدام مدير الحزم APT المشهور على أوبونتو.


الاسم على جوجل بلاي : Termux

المطور : Fredrik Fornwall

الموقع الرسمي : https://termux.com

الحجم : 17 ميغابايت

مراة التثبيت : +5M

يدعم فقط : Android 5.0 والأحدث.
